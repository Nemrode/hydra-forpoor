# hydra-forpoor

## How to use project

### Clone the project
```bash
git clone git@github.com:Nemrode/hydra-forpoor.git
```
and
```bash
cd hydra-forpoor
```

### Virtual environment
```bash
python -m venv venv
```
and
```bash
source venv/bin/activate
```

### Install dependencies
```bash
pip install -r requirements.txt
```
